# Product: DailyUp

DailyUp is an AI-powered universal learning planner. Users create learning projects in any domain, and an AI agent analyzes their goals, background, and existing skills to generate fully customized learning plans.

## Core Workflow

1. User creates a Learning Project (goal, background, skills)
2. AI Agent generates a structured Learning Plan with ordered Chapters
3. Each Chapter provides dynamically generated Learning Materials tailored to the user
4. Interactive Assessments test comprehension after each Chapter
5. AI generates Feedback Reports highlighting weak areas with review suggestions
6. Progress is tracked across all projects
7. Completion triggers a course summary and Certificate

## Key Domain Concepts

- Learning Project: user-created container with goal, background, and skills
- Learning Plan: AI-generated structured plan composed of Chapters
- Chapter: a learning unit with materials, assessments, and progress state
- User Profile: background, skills, and learning preferences
- Feedback Report: post-assessment analysis with targeted review suggestions
- Certificate: generated upon project completion

## Language

The primary product language is Chinese (Simplified). Requirements and user-facing content are written in Chinese. Code, comments, and technical documentation should be in English.
